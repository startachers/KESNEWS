    검색 및 자동화 설정

    매일 실행

    HTML을 열었을 때 당일 저장본이 없으면 자동으로 새 기사를 검색합니다.

    하루 한 번 자동 검색브라우저가 닫혀 있으면 실행되지 않습니다.

    기사 검색식

    분류별 검색식입니다. 체크를 해제하면 해당 검색은 실행하지 않습니다.

    판별 키워드

    쉼표로 구분합니다. 위험도·긍부정 분류는 제목과 기사 요약의 단어 일치에 기반합니다.

    데이터 연결

    연합뉴스 공식 RSS를 먼저 읽고 Google 뉴스 RSS로 보완하며, 수집 경로가 실패하면 GDELT 공개 기사 색인을 시도합니다.

    연합뉴스 공식 RSS 우선 수집최근 종합 뉴스에서 지정 관련도 기준과 일치하는 기사만 반영합니다.
    공개 프록시에는 검색어와 RSS 주소가 전달됩니다. 기관 내부·비공개 키워드나 API 비밀키를 이 HTML에 입력하지 마세요. 네이버 뉴스 API를 사용하려면 비밀키를 서버에 보관한 기관용 프록시 URL을 연결하는 방식을 권장합니다.
