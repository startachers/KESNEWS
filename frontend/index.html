    기사 검색식

    분류별 검색식입니다. 체크를 해제하면 해당 검색은 실행하지 않습니다.

    쉼표로 구분합니다. 위험도·긍부정 분류는 제목과 기사 요약의 단어 일치에 기반합니다.

    데이터 연결

    연합뉴스 공식 RSS를 먼저 읽고 Google 뉴스 RSS로 보완하며, 수집 경로가 실패하면 GDELT 공개 기사 색인을 시도합니다.

    연합뉴스 공식 RSS 우선 수집최근 종합 뉴스에서 지정 관련도 기준과 일치하는 기사만 반영합니다.
    국무조정실(총리실) 보도자료 직접 수집opm.go.kr 보도자료 목록을 직접 읽어옵니다.
    기후에너지환경부 보도자료 직접 수집me.go.kr 보도자료 목록을 직접 읽어옵니다.
    RSS·GDELT 수집은 이 컴퓨터의 로컬 서버(127.0.0.1:8787)가 직접 호출하며 외부 공개 프록시를 거치지 않습니다. 기관용 뉴스 API를 연결할 경우 비밀키는 서버 쪽 프록시 URL에 보관하고 이 화면에는 입력하지 마세요.
